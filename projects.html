Projects

Neurosymbolic System

  • Developed a neurosymbolic reasoning system combining the intuition of LLMs (System 2) with heuristic search over custom domain-specific libraries (System 1).
  • Implemented a causation detection mechanism to automatically identify the transformation rules leading from input to output visual grid.
  • Enhanced the system's generalization capability by implementing adaptive learning loops over linguistic queries for efficient Python code generation using the DSPy library.
  • Achieved high accuracy in the prediction of grid size, type of patterns, and count of patterns. Achieved medium-low accuracy in causation detection and replication.

    Dynamics on Network

  • Enhanced the networkx library to develop a module for the generation and handling of random, strongly connected directed graphs without isomorphic bias as observed in various natural and artificial systems.
  • Developed a network simulator with adaptive agents at each node, exploring their internal dynamics over algorithmic space during synchronous gameplay.
  • Deduced the stabilization and emergence criteria for signals of agents from an evolutionary game theory perspective.
    •

    Random Graphs

  • Discovered two algorithms for generation of strongly connected, edge-critical, directed graphs. Found all such graphs for 2-10 nodes.
  • Discovered an algorithm to generate all strongly connected, edge-critical, directed graphs of N+1 nodes given the collection of all such graphs of N nodes. Proved the theoretical correctedness of the algorithm.
  • Based the above-mentioned result, found an algorithm to generate random, strongly connected directed graphs without isomorphic bias. This was used in the Dynamics on Network project.
    •

    Cellular Automata

  • Simulated binary cellular automata on topologies like ring, hypercube, clique, etc to detect and study the emergence of complex patterns using all first neighbour binary evolution rules.
  • It was found that unlike a 1D-ring (where Rules 30 and 110 lead to emergent structures), other rules also lead to emergent behaviour on different network topologies.
